# Taplink лендинг для Instagram

Мобильный лендинг для продажи программы «Формула безопасной сушки».

## Что уже реализовано
- Полностью адаптивная мобильная структура (single-page).
- Блоки: оффер, боли, решение, наполнение, результат, соцдоказательства, тарифы, оплата.
- 2 тарифа с отдельными кнопками оплаты.
- Контент вынесен в отдельный файл `content.js`, чтобы удобно редактировать через GitHub.

## Как редактировать тексты, цены, фото и кнопки оплаты
Открой файл `content.js` и меняй значения:
- `hero`, `painPoints`, `solution`, `inside`, `result` — тексты блоков.
- `testimonials`, `beforeAfter` — фото отзывов и до/после.
- `tariffs` — названия, цены, списки, кнопки и **ссылки оплаты** (`paymentUrl`).

## Важно по YooKassa
Сейчас в шаблоне стоят плейсхолдер-ссылки для оплаты:
- `https://yoomoney.ru/quickpay/form/placeholder-solo`
- `https://yoomoney.ru/quickpay/form/placeholder-club`

Замени их на реальные ссылки оплаты YooKassa/ЮKassa для каждого тарифа.

> Если нужна оплата именно через API YooKassa (с созданием платежа на лету), понадобится backend
> (например, на Node.js/Python/PHP) для безопасной работы с секретным ключом.

## Локальный запуск
Открой `index.html` в браузере или запусти простой сервер:

```bash
python3 -m http.server 8080
```

И открой `http://localhost:8080`.
